# 超强音形用户词库（默认为加的词与原码表所带的重码时，自加词居后，若为空码，则自加词补全空码）
# 音形是超强两笔的26键变种，用的是键盘选重。e是次选，i是三选，o四选，u第五重，v为第六重。这是码表自带的五码，所以四码重码够5～6个时，不建议再加重。
# 词条、编码、权重三者之间，前者是制表符，后者才是空格，，你也可以复制黑括号中的制表符【	】使用。或者使用本键盘的〈I〉功能键盘，使用上面的Tab键。
# 如两个用户词重码，则需加权重排序，加权方式有两种，现在只说一种，另一种没必要说明，如下：
# 同文	tgwy i
# 同文	tgwyi
# 因为在原始码表里面，
# tgwy→图文（首词条）、眺望（第二词条，用第五码e选重）。所以往后排，同文为第三重码、需用i选重。
# 所以结合此码表用eiouv选重的特点，将同文一词加权的时候需如上操作
# tgwy i中tgwy与i之间是空格，用来给系统读取逐渐键位提示用。tgwyi才是真的第五码选重。。
